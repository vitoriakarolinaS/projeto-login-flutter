## 📌 Projeto

Aplicação mobile desenvolvida em Flutter com foco em simular um sistema de autenticação de usuários, incluindo cadastro, login e controle de sessão.

O projeto foi estruturado seguindo boas práticas de organização de código, separando responsabilidades em camadas.

---

## 🚀 Funcionalidades

✔ Tela de Splash  
✔ Cadastro de usuário  
✔ Login com validação  
✔ Validação de email e senha  
✔ Bloqueio de usuários duplicados  
✔ Controle de usuário logado  
✔ Listagem de usuários cadastrados  
✔ Logout  

---

## 🧠 Arquitetura

O projeto segue uma estrutura organizada em camadas:

- **Models:** representação dos dados  
- **Data:** armazenamento em memória (mock)  
- **ViewModels:** regras de negócio  
- **Views:** interface do usuário  

---

## 📂 Estrutura do Projeto
